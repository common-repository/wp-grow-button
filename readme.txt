== Description ==
"Grow!" is the online platform which helps publishers monetize directly through a button.
Publishers can be support financially and their post about their contents can be shared by users.

This plug-in makes it easy to add the Grow! button to wordpress blogs.

== Installation ==

1. Upload this directory to the 'wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open 'WP Grow Button Options' control panel through the 'Settings' menu and change configurations (This step is not required, the plugin has default settings)
